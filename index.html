The Real-Time Stock Market Simulator is a Java-based application designed to simulate the basic functioning of a stock trading system. The project allows users to buy and sell stocks whose prices change dynamically in real time. Stock price fluctuations are simulated using background threads to mimic real market behavior. The system maintains a user portfolio that tracks available balance, owned stocks, and profit or loss after each transaction. Core object-oriented programming concepts such as classes, objects, encapsulation, and inheritance are used to model real-world entities like stocks, market, and portfolio. The project provides a practical understanding of multithreading, collections, and real-time data handling, making it a realistic and educational simulation of stock market trading.
